# Generation Rules

## 基本约束

始终满足以下要求：

1. 只出单选题
2. 每题只能有一个正确答案
3. 题干、选项、答案、解析必须完整
4. 解析必须能独立解释为什么正确、为什么其余选项不正确
5. 选项长度、语气、层级尽量均衡

## 真实考试特征（融入真题风格）

根据2024年事业单位考试真题分析，真实考题有以下特征：

### 题干特征
- **公基**：题干通常较短（30-50字），直接考知识点，少有复杂语境
- **职测言语**：题干较长（80-150字），有完整语段，侧重理解
- **职测判断**：题干给出明确定义或条件，需严格比对
- **数量/资料**：数据关系清晰，口径明确

### 常见陷阱选项

**公基常见陷阱**：
| 陷阱类型 | 错误示例 | 正确特征 |
|---------|---------|---------|
| 主体错误 | 把"人民法院"混淆为"人民检察院" | 严格对应法律执行主体 |
| 程度错误 | "必须""一切""所有"等绝对化表述（除非考的就是绝对化） | 使用"一般""通常""通常情况下" |
| 范围错误 | 把"行政处罚"扩大为"刑事处罚" | 准确区分不同责任类型 |
| 时间错误 | 把"应当"说成"可以"，或混淆时效 | 准确表述权限范围 |

**职测常见陷阱**：
| 陷阱类型 | 错误示例 | 正确特征 |
|---------|---------|---------|
| 相关≠因果 | 把相关关系当作因果关系 | 严格区分逻辑关系 |
| 表面相似 | 选项结构相同但语义不同 | 选项平行，语义各异 |
| 以偏概全 | 举例说明≠全面概括 | 整体把握文段主旨 |
| 偷换概念 | 改变限定词或范围 | 严格对应原文表述 |

### 真实考试出题偏好（近年趋势）

**公基高频考点**：
- 宪法：公民基本权利、国家机构职权分工、宪法修正案时间线
- 民法典：民事法律行为效力、合同履行、物权变动
- 行政法：行政处罚种类、行政许可与行政许可对比
- 政治：唯物辩证法基本规律、社会主义矛盾理论
- 经济：市场经济基本特征、宏观调控手段、财政货币政策区分

**职测高频考点**：
- 言语：选词填空（语境辨析）、主旨概括、语句排序
- 判断：定义判断（多定义）、类比推理（内涵关系）、逻辑判断（加强/削弱）
- 数量：工程问题、行程问题、利润问题、排列组合
- 资料：增长率计算、比重变化、平均数比较

## 仿题约束

学习参考题时，允许复用"结构"，不允许复用"表面内容"。

### 可以复用

- 子题型
- 设问方式
- 解题路径
- 干扰项设计思路
- 难度层级

### 不可以复用

- 原题事实材料
- 原题表述骨架
- 原题答案位置
- 原题专有名词、数字、场景组合
- 原题解析句式

## 自检清单

输出前逐项检查：

### 答案唯一性

- 是否存在两个以上看似都正确的选项
- 是否存在题干限定不足导致多解
- 是否有选项因为表述绝对化而显得异常突兀

### 选项平衡性

- 语法结构是否大致一致
- 长短是否差距过大
- 是否出现一个选项明显更具体或更空泛

### 解析闭环

- 是否指出核心考点
- 是否给出正确答案成立的理由
- 是否至少解释两个错误选项的问题

### 风格一致性

- 是否符合对应考试类型和地区的常见风格
- 是否超出用户指定的难度等级
- 是否引入了主观题表达或开放式答案

### 历史事实准确性（历史文化、科技人文模块必检）

**四大发明时间线（高频考点，易错）**：
| 发明 | 正确时间 | 常见错误 |
|------|---------|---------|
| 造纸术 | **西汉**（甘肃放马滩） | 误说东汉 |
| 印刷术 | **唐代**（雕版印刷）/**北宋**（毕昇活字） | 混为一谈 |
| 火药 | 唐代（炼丹发现）→ **宋代**军事应用 | 误说唐代用于军事 |
| 指南针 | **宋代**（航海应用）/**战国**（司南原型） | 误说宋代之前航海 |

**其他常见历史错误**：
- 商鞅变法：公元前356年（不要误说成其他时间）
- 科举制度：隋炀帝大业元年（607年）首创
- 三省六部制：隋唐确立（不要误说成三国或之前）
- 秦始皇统一六国：公元前221年

若不确定历史时间，**不要出题，宁可换其他考点**。

## 难度控制

### 基础

- 题干短（公基30字内，职测50字内）
- 信息点少（1-2个）
- 干扰项弱（明显错误）
- 计算量低或无需计算

### 中等

- 题干适中（公基50字内，职测100字内）
- 信息量适中（2-3个）
- 干扰项具备迷惑性（需认真比较）
- 需完整走解题路径

### 提高

- 题干较长（公基50-80字，职测100-150字）
- 材料更密（3-4个信息点）
- 干扰项更像正确答案（易错选）
- 需两步以上判断或计算

## 模块特殊要求

### 言语

- 不用生僻词压难度
- 主要通过语境和逻辑压难度
- 选词填空要给出完整语境

### 判断

- 不靠文字陷阱制造假难
- 主要通过要件、关系和论证结构压难度
- 定义判断必须给出完整定义

### 数量

- 不靠超长运算压难度
- 主要通过建模、设元、关系转换压难度
- 答案要整除或为整数

### 资料

- 口径、单位、年份必须清楚
- 优先估算思路，不堆复杂公式
- 答案通常是简单整数或一位小数

### 公基

- 不出强时政依赖题
- 不出需要最新政策口径才能判断的题
- 避开各地考试差异过大的地方性法规
- 优先使用全国统一适用的法律条款

### 图形推理（ASCII版）

**支持题型**（可用ASCII字符实现）：

| 类型 | 示例 | 可行性 |
|------|------|--------|
| 位置规律 | 符号平移、旋转、翻转 | ✅ 可做 |
| 样式规律 | 加减同异、黑白运算 | ✅ 可做 |
| 数量规律 | 点线面数量变化 | ✅ 可做 |
| 简单组合 | 符号叠加、替换 | ✅ 可做 |

**不支持题型**（无法用ASCII实现）：

| 类型 | 原因 |
|------|------|
| 对称性判断 | 需要视觉判断图形是否对称 |
| 空间重构 | 立方体展开图、折叠图 |
| 复杂组合图形 | 超出ASCII表达能力 |

**出题规则**：

1. 使用ASCII字符（○ □ △ ◇ ☆ ★ ■ □ ◆ ● ◆ ◎ █ 等）组成图形
2. 题干要清晰说明图形排列方式和规律类型
3. 选项用同等ASCII字符表示选项图形
4. 解析要详细说明规律判断过程

**ASCII图形题示例**：

```
【图形推理】观察规律，选择正确答案：

九宫格型：
第1行：○○△  △○□  □△○
第2行：☆☆◇  ◇☆☆  ◆◇☆
第3行：○○☆  ☆○○  ？

A. ◇☆○  B. ○◇☆  C. ☆○◇  D. ◇○☆
```

**输出时必须附带以下备注**：

```
⚠️ 图形推理说明
本题目采用ASCII字符形式，不涉及图形显示。
本形式可覆盖：位置规律、样式规律、简单数量规律。
未覆盖题型（需用户自行练习）：
  • 对称性判断（轴对称、中心对称）
  • 空间重构（立方体展开图、折叠）
  • 复杂组合图形
如需练习此类题型，请使用真实图形题库。
```

**出题时判断**：
- 用户明确要求"图形推理"时，优先出ASCII版本
- 用户提到"空间重构"或"对称"时，明确告知无法出此类题，建议用户使用真实题库

## 失败时的处理顺序

1. 先重写选项
2. 再重写题干
3. 再调整答案与解析
4. 仍无法保证唯一答案时，放弃该题并重出
